Инструкция по установке и эксплуатации

Этот раздел закрывает критерии по аппаратным требованиям, установке, настройке и эксплуатации программного обеспечения.

Минимальные требования

CPU 2 ядра, RAM 4 GB, Disk 1 GB, Python 3.11+, браузер Chrome/Edge/Firefox/Safari.

Установка

  1. Открыть папку проекта.
  2. Создать виртуальное окружение: python -m venv venv.
  3. Активировать окружение.
  4. Установить зависимости: pip install -r requirements.txt.
  5. Создать базу: python manage.py migrate.
  6. Заполнить каталог: python manage.py seed_demo.
  7. Запустить сервер: python manage.py runserver.

Эксплуатация

  1. Перейти на главную страницу.
  2. Открыть каталог и выбрать категорию.
  3. Добавить товар в корзину или избранное.
  4. Открыть корзину и проверить итоговую сумму.
  5. Зарегистрироваться через форму с Email.
  6. Отправить сообщение через страницу контактов.
  7. Переключить тему кнопкой 🌗.

Схема работы

ШагКомпонентРезультат
1URLПользователь открывает страницу
2ViewDjango получает данные из модели или сессии
3TemplateHTML-страница отображается в браузере
4JSТема, фильтр, слайдер и подтверждения работают на клиенте
