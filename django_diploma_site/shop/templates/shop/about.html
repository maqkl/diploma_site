About

О проекте

TechMarket - учебный Django-сайт для дипломной работы. Проект демонстрирует работу с базой данных, шаблонами, формами, сессиями и JavaScript-интерфейсом.

Цель - показать законченное веб-приложение с понятной архитектурой и удобным интерфейсом.

Что реализовано

Шапка, логотип, навигация, подвал на всех страницах, карточки товаров, форма, таблицы, адаптивность, гиперссылки и подсказки.

Наведите сюда, чтобы проверить всплывающую подсказку.

Frontend

HTML, CSS внутри шаблона, минимальный внешний CSS, чистый JS.

Backend

Django views, models, forms, admin, sessions, auth.

Database

SQLite для локального запуска и демонстрации.
